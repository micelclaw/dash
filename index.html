La aplicación no arrancó

El navegador cargó la página pero el bundle de JavaScript no completó el arranque. Suele ser una extensión que bloquea módulos, una caché del Service Worker en mal estado, o un error de red cargando una pieza del código.

  • Reintenta con una ventana de incógnito o un perfil limpio.
  • Desactiva extensiones que actúen en todas las páginas y recarga.
  • Limpia caché y datos del sitio (DevTools → Application → Storage → Clear site data).
